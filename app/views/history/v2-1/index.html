Check the MOT history of a vehicle

See the results of a vehicle's MOT tests. Find out:

  • if a vehicle passed or failed its MOT
  • where its MOT tests were done
  • when its next MOT is due
  • mileages recorded at the time of testing
  • advisory and failure items from each test

You’ll see the results for a new MOT as soon as the test is done.

Start now

Before you start

You’ll need the vehicle’s registration number (number plate).

You can only get test results from 2005 onwards, for tests done in England, Scotland or Wales.
